## Description
This project is the backend API for a Pinterest-style image sharing application. It handles user authentication, image and video uploads, board and pin management, likes, comments, and follow functionality.

The backend is built with Node.js, Express, and MongoDB (Atlas), and uses Cloudinary for media storage.

## Live API
https://pinterest-backend-088x.onrender.com

## Tech Stack
- Node.js
- Express
- MongoDB (Atlas)
- Cloudinary
- JWT Authentication
